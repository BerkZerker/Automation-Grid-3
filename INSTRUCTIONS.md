# How to Run Automation Grid 3 MVP

## Prerequisites
- Godot 4.5 (or latest 4.x)

## Running the Game
1. Open the project in Godot.
2. Run the project (F5). The **Main Menu** will appear.

### Multiplayer Testing
To test multiplayer locally:
1. Go to **Debug** > **Run Multiple Instances** in the Editor menu.
2. Set it to **2 instances**.
3. Run the project.
4. On Instance 1: Click **Host Game**.
5. On Instance 2: Click **Join Game** (Default IP 127.0.0.1 is pre-filled).

## Controls
- **WASD:** Move
- **Space:** Jump
- **Left Click:** Place Building (looks like a generic box)
- **Mouse:** Look around

## Features
- **Procedural Terrain:** Infinite terrain generates around you.
- **Building:** You can place factories on the terrain.
- **Automation:** Placed factories generate "Iron Ore" every second (counter on top).
